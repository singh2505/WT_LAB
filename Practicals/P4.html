HTML Tags and their attributes

  • Body -
    • TEXT - This attribute allows you to set the color of the text.
    • BGCOLOR - This attribute allows you to set the color of the background.
    • BACKGROUND - This attribute allows you to set an image as the background.
  • Form -
    • METHOD - This attribute allows you to set the method of form submission.
    • ACTION - This attribute allows you to set the action after form submission.
  • Img -
    • SRC - This attribute allows you to set the path of image.
    • WIDTH - This attribute allows you to set the width of image.
    • HEIGHT - This attribute allows you to set the height of image.
  • A -
    • HREF - This attribute allows you to set the url of location.
    • NAME - If the NAME attribute is included instead of the HREF attribute then the A tag is being used to mark a spot in the page that can later be pointed to by a link.
  • Table -
    • BORDER - This defines how big the borders of a table will be.
    • WIDTH - This defines the width of the table.
    • BGCOLOR - This defines the background color of the table. If a color is specified in the TR or TD tags BGCOLOR is ignored for that row, or cell respectively.
    • CELLPADDING - This defines how far from the edge of a cell the content in that cell will be.
    • CELLSPACING - This defines how far apart the cells of the table will be.
